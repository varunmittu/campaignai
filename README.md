# CampaignAI 🚀
> AI-powered full marketing campaign generator — built with React + Claude API

## What it does
Fill a 3-step brief → get a complete campaign:
- ✅ Campaign headline, tagline, hook & positioning
- ✅ 3 ad copy variants (Awareness / Conversion / Retargeting)
- ✅ Platform-specific social media posts
- ✅ Full 3-email nurture sequence
- ✅ 3 content ideas with scripts
- ✅ 4-week strategy, KPIs & budget allocation

## Deploy in 5 Minutes

### Step 1 — Upload to GitHub
1. Go to https://github.com → click **New Repository**
2. Name it `campaignai` → click **Create Repository**
3. Open your terminal and run:
```bash
cd campaignai
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/campaignai.git
git push -u origin main
```

### Step 2 — Deploy on Vercel
1. Go to https://vercel.com → Sign in with GitHub
2. Click **Add New Project**
3. Select your `campaignai` repository → click **Import**
4. Framework will auto-detect as **Vite**
5. Click **Deploy** — done! ✅

> The `vercel.json` file is already included — it fixes the 404 routing issue automatically.
